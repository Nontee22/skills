# Skill 清单与用途说明

## 一、简历改写类

### 1. ai-resume-polish — AI/Agent 方向简历改写
将流水账式的 AI/大模型/Agent 项目描述转化为面试官想看的结构：问题驱动 + 技术方案 + 量化成果。覆盖多 Agent 架构、RAG 知识增强、模型微调、Prompt 工程、MCP 工具接入、LLM 评测等场景。

### 2. java-resume-polish — Java 后端简历改写
将 Java 后端项目的功能罗列转化为体现架构思考和技术深度的简历描述。覆盖高并发性能优化、分布式系统、数据库优化、缓存设计、消息队列、微服务架构、稳定性建设等场景。

---

## 二、文章处理类

### 3. article-learning-pipeline — 通用技术文章学习管道
两阶段学习材料管道：(1) 将原始技术文章（Vue/小程序/AI Agent/Java 等）提取为带来源追溯的 P-xxx 学习卡片；(2) 按「建议学习顺序」拆分章节、分派 Worker 并行处理、合并为最终学习文档。

### 4. java-article-pipeline — Java 技术文章学习管道
与通用版同架构，额外包含 Java 领域分类体系（java-taxonomy.md），针对 Java 技术栈（并发、JVM、框架源码等）做更精准的知识点提取和深挖提示卡生成。

### 5. article-readability-enhancer — 文章可读性增强器
在不丢失任何原文内容的前提下，对文章进行结构重组、表达优化和排版增强。拆分长段落和长句、添加标题层级、显式化逻辑关系、添加提示框，使文章层次分明、适合新手阅读。核心铁律：一字一句都不能丢。

## 三、设计品味工具集（design-taste-toolkit）

### 6. design-direction-inference — 设计方向推断引擎
反模板化前端设计技能。读取用户需求简报，推断设计方向（受众、风格、参考信号），设定三个核心旋钮（设计变体度、动效强度、视觉密度），选择设计系统或美学风格，生成不显模板痕迹的高品质落地页/作品集/品牌站。

### 7. editorial-minimal-ui — 编辑风极简 UI
温暖单色调色板 + 排版对比 + 扁平便当网格 + 柔和粉彩点缀。无渐变、无重阴影，适合文档风格、知识库、极简品牌站等场景。

### 8. premium-brand-ui — 高端品牌 UI
代理商级别数字体验设计。双边框嵌套架构、流体动效编排、磁吸悬停物理、毛玻璃浮岛导航，适合高端 SaaS、奢侈品牌、精品工作室等场景。

### 9. elite-motion-design — 精英动效设计（Awwwards 级）
Python 驱动真随机化布局 + AIDA 结构 + 2 行标题铁律 + 无空隙便当网格 + GSAP ScrollTrigger 高级编排（钉住/堆叠/擦除），适合冲击奖项的创意型落地页。

### 10. industrial-terminal-ui — 工业终端 UI
融合瑞士印刷排版与军用终端美学的原始机械感界面。刚性网格、极端字号对比、实用主义配色、模拟退化效果（网版、扫描线、抖动），适合数据密集型仪表盘或“解密蓝图”风格站点。

### 11. motion-craft-engine — 动效工艺引擎
编码 Emil Kowalski 的 UI 工艺哲学：动画决策框架（频率 → 目的 → 缓动 → 速度）、Spring 物理动画、组件构建原则（按钮反馈、弹窗原点感知、提示框延迟跳过）、性能规则（仅变换和透明度、WAAPI），让界面“感觉对”。

### 12. animation-reviewer — 动画审查器
审查动效代码是否符合高标准工艺。十条不可妥协标准（动画合理性、频率适配、缓动响应、300ms 上限、原点正确性、可中断性、GPU 属性、无障碍、进出不对称、整体协调），输出问题表 + 裁决（通过/阻止）。

### 13. animation-glossary — 动画术语词典
反向查找词典：将模糊的动效描述转为精确术语（“弹窗打开时弹一下那个效果”→ Pop in，“iOS 橡皮筋滚动”→ Rubber-banding），帮用户用正确的词去追问 AI 或沟通设计师。

### 14. find-skills — 技能发现器
帮助用户搜索和安装开放 Agent Skills 生态中的技能包。支持关键词搜索、质量验证（安装量/来源/星数）、一键安装，覆盖 Web 开发、测试、DevOps、文档、代码质量、设计、生产力等类别。
